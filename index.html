🎮 How to Play

  • First install Minecraft
  • Open Minecraft
  • Create/Login into your account
  • Go to Multiplayer section
  • Add server information (given in server IPs)
  • Join and play

🕹️ How to Start Server (If Offline)

  • Click on Start Server (opens the public server page)
  • If it shows offline, click the Start button
  • If it shows expired, click Renew button
  • Choose Normal Renew
  • If it doesn't ask for renew type, just click Renew
  • Then click Start button — Server will start in 5 mins
