📘 How to Play

  • 1. First install Minecraft
  • 2. Open Minecraft
  • 3. Create/Login into your account
  • 4. Go to Multiplayer section
  • 5. Add server information (see Server Info above)
  • 6. Join and play 🎮

🚀 How to Start Server If It Is Offline

  • 1. Click on Start Server above (opens server public page)
  • 2. If it shows Offline, click the Start button
  • 3. If it shows Expired, click the Renew button
  • 4. Choose Normal Renew
  • 5. If Normal Renew doesn’t appear, click Renew then click Start
  • 6. Server will start in about 5 minutes
